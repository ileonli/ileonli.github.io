Skip List

LinkedList

在讨论跳表前，我们先回顾一下链表（LinkedList），链表有以下优点和缺点：

  • 链表插入和删除给定的节点，只需要更改指针的指向位置即可，因此空间复杂度为 O(1)。
  • 对于搜索操作，由于链表是无序的（无法使用二分查找），只能从头部搜索到尾部，空间复杂度为 O(n)。

为了加快链表搜索过程，跳表引入了分层的概念。跳表的每一层都是一个有序链表，且最底层是初始的有序链表（包含所有元素）。每个位于第 i 层的节点有 p 的概率出现在第 i+1 层，p 为常数。

跳表的多层索引机制在逻辑上模仿了二分查找的分治策略，使得链表也能实现对数时间复杂度的查找。

Introduction

跳表是一种可以替代平衡树的数据结构。跳表使用概率平衡，而不是严格的平衡维护，因此，跳表中插入和删除的算法比平衡树中的等价算法要简单得多，速度也显著更快。通过在链表上添加索引，达到优化查询的目的。

概率算法（Probabilistic Algorithm）：算法的执行过程中使用随机数来影响决策或流程的算法。这意味着即使对于相同的输入，算法每次运行的结果或执行路径也可能不同。

每个元素在跳表中由一个节点表示（如下图所示），节点是以 key 的顺序有序排列的。且每个节点都有一个高度（height）或级别（level），表示该节点拥有的前向指针的数量。

当插入节点时，会随机生成节点的高度，而不需要考虑当前跳表中的元素数量。

对于上图中的链表，我们可以发现下边的规律：

  • 第 0 层：这是完整的链表，包含所有按顺序排列的元素。
  • 第 1 层：这一层包含第 0 层中大约一半的元素。通过从第 0 层中随机选择大约一半的节点，并将它们链接到第 1 层。
  • 第 2 层：这一层包含第 1 层中大约一半的元素，意味着它包含第 0 层总元素数的 1/4 左右。
  • 第 3 层：这一层包含第 2 层中大约一半的元素，大约是第 0 层总元素数的 1/8，以此类推。

数据结构

Node 类

使用 Node 类来表示跳表中的最底层的节点，该类有以下属性：

  • Node 使用 K key 和 V value 分别表示键和值。
  • Node<K, V>[] forwards 表示该节点上的索引，该数组的初始长度为跳表的最大高度（DEFAULT_MAX_LEVEL），如果 forwards[i] 指向 null，表示第 i 层不指向下一列（没有索引）。
public class Node<K extends Comparable<K>, V> {
    K key;
    V value;

    Node<K, V>[] forwards;

    @SuppressWarnings("unchecked")
    public Node(K key, V value, int level) {
        this.key = key;
        this.value = value;
        this.forwards = new Node[level]; // level 为跳表的最高高度
    }
}

SkipList

使用 SkipList 类来表示跳表，该类有以下属性：

  • Node<K, V> head：该头节点不存储键值，只存储指向下一列的引用，通过 head.forwards[0] 即可获取第一列的第 0 层节点。
  • int level：用来表示当前跳表中最高一列的高度（需保证 level < DEFAULT_MAX_LEVEL），初始为 0，表示没有节点。

Search

每次查找都是从最高层索引逐渐向低层查找，向下查询时有两个路径：

  1. 如果当前节点的下一列的 key 小于 target，则表示要查找的节点在下一列的右边，向右查找。
  2. 如果当前节点的下一列的 key 大于等于 target，则表示要查找的节点在下一层，向下查找。

public V search(K key) {
    if (key == null) {
        return null;
    }

    Node<K, V> current = head;
    for (int i = level - 1; i >= 0; i--) {
        while (current.forwards[i] != null && less(current.forwards[i].key, key)) {
            current = current.forwards[i];
        }
    }

    // 此处的 current 一定位于第 0 层，且 current.key >= key

    // 获取下一列的第 0 层的节点
    current = current.forwards[0];

    if (current != null && equal(current.key, key)) {
        return current.value;
    }

    return null;
}

Insert

插入操作需要先找到要插入位置的前一个节点，把新节点放入在该节点的后一列即可。使用 update 数组记录每一层中小于待插入键的第一个节点，用于此列建立索引。

使用 newLevel = randomLevel() 生成新插入节点的高度：

  • 如果 newLevel 高于当前跳表，需要先把高于当前跳表的节点指向尾节点（null）。为方便操作，可先把 update[i] 指向 head，后续使用 update[i].forward[i] 获取尾节点。
  • 如果 newLevel 低于当前跳表，把 update 中高度小与等于 newLevel 的节点的 forward 进行更新。插入节点的左边一列的 forwards 指向新节点，插入节点的 forwards 指向右边一列。

public V insert(K key, V value) {
    Objects.requireNonNull(key, "key");

    Node<K, V>[] update = (Node<K, V>[]) new Node[DEFAULT_MAX_LEVEL];
    Arrays.fill(update, head);

    Node<K, V> current = head;
    for (int i = level - 1; i >= 0; i--) {
        while (current.forwards[i] != null && less(current.forwards[i].key, key)) {
            current = current.forwards[i];
        }
        update[i] = current;
    }

    if (current.forwards[0] != null && equal(current.forwards[0].key, key)) {
        V oldValue = current.forwards[0].value;
        current.forwards[0].value = value;
        return oldValue;
    }

    int newLevel = randomLevel();
    Node<K, V> newNode = new Node<>(key, value, DEFAULT_MAX_LEVEL);
    for (int i = 0; i < newLevel; i++) {
        newNode.forwards[i] = update[i].forwards[i];
        update[i].forwards[i] = newNode;
    }

    this.level = Math.max(this.level, newLevel);

    return null;
}

Delete

删除操作需要先找到要删除的节点，并使用 update 数组记录存储了每一层中小于键的最后一个节点。

如果找到要删除的节点，遍历所有高度小与等于删除节点的节点，把上一个节点的 forwards 指向删除节点的 forwards 即可。

在删除跳表中高度最高的节点后，可能导致某些高层级变为空的节点后，需要适当降低跳表的最高层级。

删除节点 16 后，最高层就变空了，我们需要将当前跳表高度降低一层以去除空层

public V remove(K key) {
    if (key == null) {
        return null;
    }

    Node<K, V>[] update = (Node<K, V>[]) new Node[DEFAULT_MAX_LEVEL];

    Node<K, V> current = head;
    for (int i = level - 1; i >= 0; i--) {
        while (current.forwards[i] != null && less(current.forwards[i].key, key)) {
            current = current.forwards[i];
        }
        update[i] = current;
    }

    current = current.forwards[0];

    if (current == null || !equal(current.key, key)) {
        return null;
    }

    for (int i = 0; i < level; i++) {
        if (update[i].forwards[i] != current) {
            break;
        }
        update[i].forwards[i] = current.forwards[i];
    }

    while (level > 1 && head.forwards[level - 1] == null) {
        level--;
    }

    return current.value;
}

Range Search

如果需要查找 10 <= k <= 100 之间的节点，只需要先找到 10 所在的节点，沿着 forwards[0] 一直找到第一个大于 100 的节点停止。

Implementation

import java.util.Arrays;
import java.util.Objects;
import java.util.Random;

public class SkipList<K extends Comparable<K>, V> {

    public static class Node<K extends Comparable<K>, V> {
        final K key;
        V value;

        Node<K, V>[] forwards;

        @SuppressWarnings("unchecked")
        public Node(K key, V value, int level) {
            this.key = key;
            this.value = value;
            this.forwards = new Node[level];
        }

        public K getKey() {
            return key;
        }

        public V getValue() {
            return value;
        }

        public void setValue(V value) {
            this.value = value;
        }
    }

    private static final int DEFAULT_MAX_LEVEL = 32;
    private static final double DEFAULT_PROBABILITY = 0.5;

    private final Node<K, V> head;

    private int level;

    private final Random rand = new Random();

    public SkipList() {
        this.head = new Node<>(null, null, DEFAULT_MAX_LEVEL);
        this.level = 0;
    }

    public V search(K key) {
        if (key == null) {
            return null;
        }

        Node<K, V> current = head;
        for (int i = level - 1; i >= 0; i--) {
            while (current.forwards[i] != null && less(current.forwards[i].key, key)) {
                current = current.forwards[i];
            }
        }

        current = current.forwards[0];

        if (current != null && equal(current.key, key)) {
            return current.value;
        }

        return null;
    }

    @SuppressWarnings("unchecked")
    public V insert(K key, V value) {
        Objects.requireNonNull(key, "key");

        Node<K, V>[] update = (Node<K, V>[]) new Node[DEFAULT_MAX_LEVEL];
        Arrays.fill(update, head);

        Node<K, V> current = head;
        for (int i = level - 1; i >= 0; i--) {
            while (current.forwards[i] != null && less(current.forwards[i].key, key)) {
                current = current.forwards[i];
            }
            update[i] = current;
        }

        if (current.forwards[0] != null && equal(current.forwards[0].key, key)) {
            V oldValue = current.forwards[0].value;
            current.forwards[0].value = value;
            return oldValue;
        }

        int newLevel = randomLevel();
        Node<K, V> newNode = new Node<>(key, value, DEFAULT_MAX_LEVEL);
        for (int i = 0; i < newLevel; i++) {
            newNode.forwards[i] = update[i].forwards[i];
            update[i].forwards[i] = newNode;
        }

        this.level = Math.max(this.level, newLevel);

        return null;
    }

    @SuppressWarnings("unchecked")
    public V remove(K key) {
        if (key == null) {
            return null;
        }

        Node<K, V>[] update = (Node<K, V>[]) new Node[DEFAULT_MAX_LEVEL];

        Node<K, V> current = head;
        for (int i = level - 1; i >= 0; i--) {
            while (current.forwards[i] != null && less(current.forwards[i].key, key)) {
                current = current.forwards[i];
            }
            update[i] = current;
        }

        current = current.forwards[0];

        if (current == null || !equal(current.key, key)) {
            return null;
        }

        for (int i = 0; i < level; i++) {
            if (update[i].forwards[i] != current) {
                break;
            }
            update[i].forwards[i] = current.forwards[i];
        }

        while (level > 1 && head.forwards[level - 1] == null) {
            level--;
        }

        return current.value;
    }

    private int randomLevel() {
        int level = 1;
        while (level < DEFAULT_MAX_LEVEL && rand.nextDouble() < DEFAULT_PROBABILITY) {
            level++;
        }
        return level;
    }

    private boolean less(K k1, K k2) {
        return k1.compareTo(k2) < 0;
    }

    private boolean equal(K k1, K k2) {
        return Objects.equals(k1, k2);
    }

}
