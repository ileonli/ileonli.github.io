表格

表格不是 Markdown 核心规范的一部分，但 Hugo 原生支持它们。

姓名年龄
Bob27
Alice23

表格内的行内 Markdown

斜体粗体代码
斜体粗体code
ABCDEF
这是一个很长的单元格内容，主要用于测试表格在不同屏幕尺寸下的响应式表现。这是另一列的内容，同样包含了大量的文字，以确保表格布局能够正确处理长文本。第三列的内容也不短，看看它会如何影响周围的列宽分配以及整体的表格结构。短文本测试中等长度的文本测试，稍微长一点最后的总结性文本，填满表格。
